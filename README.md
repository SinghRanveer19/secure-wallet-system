#  Secure Wallet System

##  Features

* JWT Authentication
* Wallet Management
* Money Transfer (Atomic Transactions)
* Idempotency Handling
* Fraud Detection System

##  Tech Stack

* Java
* Spring Boot
* Spring Security
* MySQL

##  Security

* Stateless JWT Authentication
* Password Encryption (BCrypt)

##  Key Concepts

* Transaction Management (@Transactional)
* Idempotency Keys
* Fraud Detection Rules
* Exception Handling

##  APIs

* Register
* Login
* Add Money
* Transfer Money

##  Future Enhancements

* Payment Gateway Integration
* Microservices Architecture
